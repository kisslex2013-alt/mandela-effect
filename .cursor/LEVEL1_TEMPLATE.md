# Level 1: Глобальные правила пользователя (User Rules)

## Где настраивается

Эти правила настраиваются в настройках Cursor, а не в проекте:

1. **Через UI**: Settings → Rules for AI → User Rules
2. **Через файл**: `%APPDATA%\Cursor\User\globalStorage\rules.mdc` (Windows)

## Рекомендации

- **Держите под 50 строк** - только самое важное
- **Универсальные предпочтения** - работают для всех проектов
- **Стиль общения** - как ты хочешь, чтобы AI отвечал
- **Language preferences** - язык ответов, формат

## Шаблон для копирования

```markdown
# Глобальные правила общения

## Стиль общения

- Отвечай кратко, без воды
- Используй русский язык для объяснений
- Давай конкретные решения, не общие советы
- Показывай код с комментариями на русском для сложной логики

## Предпочтения кода

- TypeScript строгий режим везде
- Всегда проверяй типы явно
- Комментарии на русском для сложной логики
- Генерируй полный рабочий код без пропусков

## Формат ответов

- Используй markdown для форматирования
- Показывай примеры кода с синтаксисом
- Объясняй "почему", не только "как"
```

## Важно

⚠️ Эти правила загружаются **всегда** для всех проектов - самый неэффективный уровень. Держите минимально!

✅ Для проектных стандартов используйте Level 2 (`.cursor/index.mdc`)
✅ Для специализированных правил используйте Level 3 (`.cursor/rules/*.mdc` с globs)
