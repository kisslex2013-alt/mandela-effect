Модели ИИ для проекта "Эффект Мандела"
Для генерации и анализа данных (LLM)
Llama 2 — лучший выбор для вашего проекта благодаря полной бесплатности для коммерческого использования. Модель с 7-70 млрд параметров доступна на HuggingFace и может быть развёрнута локально. Meta выпустила её с явным разрешением на коммерческое применение, что идеально для генерации описаний эффектов, текущего состояния, хлебных крошек и других данных.​

Falcon 40B — альтернатива с лицензией Apache-2.0, достаточно мощная и полностью открытая. Работает быстрее чем Llama на меньших объёмах текста.​

Mistral 7B — компактная модель, хорошо подходит для локального развёртывания на слабых машинах, с коммерческой лицензией.

Для генерации изображений с нуля
FLUX.1 [schnell] — абсолютно лучший выбор по качеству и скорости. Полностью открытая под Apache 2.0 лицензией, генерирует высокореалистичные изображения с точным следованием промптам и отличной обработкой текста внутри изображений. Доступна бесплатно на различных платформах (Krea.ai, Replicate, getimg.ai).​

Stable Diffusion 3.5 — матёрая экосистема с огромным количеством LoRA адаптеров. Medium версия (2 млрд параметров) более доступна, хотя коммерческое использование требует лицензии. Отличный выбор если вам нужна гибкость и customization.​

DeepFloyd IF — специализирована на сложных промптах и вставке текста в изображения. Выпущена командой за Stable Diffusion. Свободна для локального использования.​

Для обработки и редактирования изображений
ControlNet — инструмент для точного контроля над генерацией через дополнительные условия (Canny edges, OpenPose, сегментация и т.п.). Позволяет изменять фон, освещение, стиль без переобучения модели.​

IP-Adapter — для стиля и композиции, работает с любыми диффузионными моделями.

Image Upscaling — используйте бесплатные онлайн-сервисы (Pixelcut, Pixelbin ) для увеличения разрешения и изменения размера на 16:9.​

Для изменения соотношения сторон на 16:9
Leonardo AI — 150 бесплатных дневных токенов достаточно для экспериментов. Встроенная функция изменения соотношения сторон с заполнением сцены AI. Позволяет коммерческое использование на бесплатном плане.​

Getimg.ai — гибкий в выборе размеров, с множеством моделей на выбор.​

Replicate — первые 50 запросов бесплатны, затем от 1-2 центов за генерацию. Есть интеграция множества моделей в одном месте.​

⚠️ ВАЖНО: Ограничения при работе с историческими личностями
Без проблем работают (рекомендуется):
HuggingFace Diffusers — полностью открытая библиотека без встроенных ограничений. Вы можете локально запускать любые модели включая генерацию исторических фигур (Ельцин, Сталин и т.п.). Apache 2.0 лицензия.​

Локально развёрнутые модели — Stable Diffusion 1.5, SDXL, FLUX.1 [schnell], запущенные на вашей машине через ComfyUI или AUTOMATIC1111, не имеют никаких ограничений по генерации любых лиц. Это ваш основной инструмент для работы с историческими личностями.

LoRA адаптеры — можно тренировать свои LoRA для конкретных стилей или концепций. Процесс полностью локален и не регулируется платформой.​​

С проблемами (избегайте):
Civitai — с мая 2025 года запретила все LoRA и контент, основанные на реальных людях, включая исторических деятелей. Даже если человек мёртв столетия, контент удаляется.​

DALL-E 3 / ChatGPT — теперь технически позволяет генерировать публичные фигуры (с возможностью opt-out), но при этом добавляет ограничения на "сценарии с насилием" и "спорные исторические сценарии", что может заблокировать вашу работу.​

Stability AI (SD3+) — новая AUP от июля 2025 года добавила ограничения на сексуальный контент и требует проверки согласия.

Оптимальная архитектура проекта
Для анализа эффектов: Llama 2 (70B если есть GPU, иначе 7B) локально

Для генерации описаний: тот же Llama 2

Для изображений основных эффектов: FLUX.1 [schnell] на getimg.ai или Krea.ai (бесплатно)

Для исторических личностей: Stable Diffusion 1.5 или SDXL локально через ComfyUI + свои LoRA

Для стилизации под "Очень странные дела": ControlNet + IP-Adapter с промптом стиля

Для изменения аспекта 16:9: Leonardo AI или локальная обработка через скрипт Python

Все компоненты либо полностью бесплатны, либо имеют щедрые бесплатные тиры, и работают без ограничений на исторические личности при локальном развёртывании.
