# Файлы из промтов 1-4

Эта папка содержит все файлы, созданные благодаря промтам 1, 2, 3 и 4.

## Структура файлов

### ПРОМТ 1: Создание проекта
- `tailwind.config.ts` - Конфигурация Tailwind CSS с кастомными цветами и анимациями
- `app/layout.tsx` - Корневой layout с шрифтом Inter и базовыми стилями
- `app/page.tsx` - Главная страница (пустая, будет заполнена в промтах 3 и 4)
- `app/globals.css` - Глобальные стили с Tailwind директивами

### ПРОМТ 2: Создание файла данных
- `data/effects.json` - JSON файл с 15 эффектами Манделы

### ПРОМТ 3: Hero секция главной страницы
- `app/page.tsx` - Обновлен с Hero секцией (анимированный заголовок, статистика, стрелка вниз)
- `lib/hooks/useCountUp.ts` - Хук для анимации счётчика чисел
- `lib/utils/stats.ts` - Утилиты для подсчёта статистики проекта
- `app/api/stats/route.ts` - API route для получения статистики

### ПРОМТ 4: Секция "Самое спорное"
- `app/page.tsx` - Добавлена секция "Самое спорное сейчас" с карточкой эффекта
- `app/api/most-controversial/route.ts` - API route для поиска самого спорного эффекта (50/50)

## Описание компонентов

### Hero секция (ПРОМТ 3)
- Анимированный заголовок "Как ты помнишь?" с градиентом
- Подзаголовок
- Живая статистика (эффекты, участники, голоса) с анимацией счётчика
- Стрелка вниз для скролла

### Секция "Самое спорное" (ПРОМТ 4)
- Заголовок секции
- Карточка с самым спорным эффектом (ближайшим к 50/50)
- Emoji категории + название эффекта
- Вопрос эффекта
- Надписи "Вариант А" и "Вариант Б" (без самих вариантов)
- Прогресс-бар с градиентом (синий → оранжевый) и точкой-маркером
- Проценты над баром
- Общее количество голосов
- Кнопка "Посмотреть и проголосовать →"
- Fade-in анимация при скролле

## Технические детали

- **Next.js 16** с App Router
- **TypeScript**
- **Tailwind CSS 4** с кастомными цветами и анимациями
- **React Hooks** (useState, useEffect, useRef)
- **Intersection Observer** для fade-in анимации
- **API Routes** для получения данных из JSON файла
